<objective>
Add the single read query `getPollsByOrganizerId(organizerId)` to `src/lib/db/queries.ts`
and its DB-backed tests. This is the data layer for the `/polls` dashboard: it returns every
poll (open + closed) owned by one `lfg_organizer` token, newest-first, with participant-safe
columns only.

Purpose: mirror the existing `getFinalizedPollsByOrganizerId` no-leak discipline while adding
open polls and per-poll aggregate counts, so the dashboard can render a per-poll summary
without exposing any participant identity or third token.
Output: `getPollsByOrganizerId` export + tests covering ordering, counts, isolation, empty
organizer, null-organizer exclusion, and no-leak.
</objective>
