<objective>
Build the read-only data and aggregation foundation for the Results Dashboard: a shared three-state vocabulary module, a pure best-day/tally aggregation function, and an admin-only participant-safe results query. No UI in this plan — it delivers the tested, serializable building blocks that plan 03-02 renders.
