Descriptor text: Best day mode -> "Best day: {resolvedLabel} · {STATE_META[status].label}"; specific date -> "{label} · {STATE_META[status].label}"; All dates -> "All dates · {STATE_META[status].label}".

The Clear button `disabled` is DERIVED each render (not event-gated) and true ONLY when already at default (`dateSel === BEST_DAY_VALUE && status === "yes"`), so manually re-selecting the default via the dropdowns also disables it.
