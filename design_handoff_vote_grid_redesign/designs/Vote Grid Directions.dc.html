Three ways to redesign the vote grid

All three keep the shipped system verbatim — Inter, the 30/24/16/14/12 scale, 8-pt spacing, the neutral 60/30/10 base, and the exact three states Available / If-need-be / Not available with icon + text (never color alone). What changes is only the grid's mechanic and its personality dial. The participant vote screen (/p/[link]) is the hero; this board shows one open-poll row set rendered each way.
